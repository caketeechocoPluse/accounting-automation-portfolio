# 삼성전자 재무제표 분석 (2022-2024)

## 📊 프로젝트 개요
삼성전자의 최근 3개년 재무제표를 분석하여 주요 재무비율을 계산하고 시각화

## 🎯 분석 내용
- **안정성 비율**: 부채비율, 유동비율, 당좌비율
- **수익성 비율**: ROE, ROA, 영업이익률
- **추세 분석**: 3개년 비교

## 🛠 사용 기술
- Python 3.x
- Pandas (데이터 처리)
- Matplotlib (시각화)
- Jupyter Notebook
